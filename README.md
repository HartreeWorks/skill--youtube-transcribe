# YouTube Transcribe

A Claude Code skill for transcribing YouTube videos locally using Parakeet MLX, then generating summaries with chapters and timestamped quotes.

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

To use this skill with Claude Code, clone it into your skills directory:

```bash
git clone https://github.com/HartreeWorks/skill--youtube-transcribe.git ~/.claude/skills/youtube-transcribe
```

Then restart Claude Code to load the skill.
