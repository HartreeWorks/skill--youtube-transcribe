# YouTube Transcribe

A Claude Code skill for transcribing YouTube videos locally using Parakeet MLX, then generating summaries with chapters and timestamped quotes.

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

```bash
npx skills add HartreeWorks/skill--youtube-transcribe
```

If you get "command not found", [install Node](https://github.com/HartreeWorks/skills/blob/main/how-to-install-node.md) then try again.
